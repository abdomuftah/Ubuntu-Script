# Ubuntu-Script


## سكربت خاص لاصحاب سيرفرات VPS :

انسخ الامر التالي وقم بوضعه في الوحدة الطرفية :

`sudo wget https://git.io/fARE7 -O ubuntu_script.sh`


## لتشغيل السكربت :

فقط قم بنسخ الامر التالي والصقه بالوحدة الطرفيه :

`sh ubuntu_script.sh `
